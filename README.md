## 📌 Project Overview
**Veriffy** is a modern web application designed to verify information quickly and efficiently.  
The platform helps users check and validate inputs using a clean, interactive, and user-friendly interface.

The goal of this project is to create a **fast, responsive, and visually appealing verification system** that demonstrates modern web development concepts.

---

## 🚀 Features
- 🌙 Light and Dark Mode Toggle  
- ✔️ Smart Verification System  
- ⚡ Fast and Responsive Interface  
- 🎨 Modern UI Design  
- 📱 Mobile Friendly Layout  
- 🔍 Real-time Input Checking  
- ✨ Smooth Animations and Interactive Elements  

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the web application  
- **CSS3** – Styling and UI design  
- **JavaScript** – Functionality and interactivity
